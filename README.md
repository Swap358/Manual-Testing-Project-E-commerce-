# Manual Testing Project – E-commerce Web Application

This project demonstrates complete manual testing of an E-commerce web application.  
The goal of this project is to showcase real-world testing skills such as test planning, test case design, defect identification, and test documentation.

---

## Project Objective

To perform functional testing of an E-commerce website and validate major user workflows such as login, product search, cart functionality, and checkout process.

---

## Testing Scope

The following modules were tested:

- User Registration
- Login Functionality
- Product Search
- Product Details Page
- Add to Cart
- Remove from Cart
- Checkout Process
- Order Confirmation
- Basic UI Validation

---

## Testing Approach

The testing was performed using manual testing techniques.

The following types of testing were covered:

- Functional Testing
- Positive Testing
- Negative Testing
- Boundary Value Testing
- UI Validation
- Test Case Design
